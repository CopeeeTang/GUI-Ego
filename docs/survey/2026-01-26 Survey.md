 ★ Insight ─────────────────────────────────────                                                                                                                                           
  Generative UI 的核心范式转变：传统 UI 是"开发者预先设计 → 用户被动使用"，而 Generative UI 是"AI 根据意图实时生成 → 用户主动协作"。这不仅是技术变革，更是人机交互哲学的根本转变。          
  ─────────────────────────────────────────────────

  一、核心协议生态                                                                                                                                                                          
                                                                                                                                                                                            
  1. Google A2UI (Agent-to-User Interface)                                                                                                                                                  
                                                                                                                                                                                            
  发布时间：2024年12月 / 2025年1月开源                                                                                                                                                      
                                                                                                                                                                                            
  核心定位：让 AI Agent 安全地生成富交互 UI 的声明式协议                                                                                                                                    
                                                                                                                                                                                            
  技术架构：                                                                                                                                                                                
  Agent 生成 JSON → 声明式 UI 描述 → 客户端原生渲染                                                                                                                                         
                                                                                                                                                                                            
  关键特性：                                                                                                                                                                                
  ┌─────────────┬───────────────────────────────┐                                                                                                                                           
  │    特性     │             说明              │                                                                                                                                           
  ├─────────────┼───────────────────────────────┤                                                                                                                                           
  │ 声明式 JSON │ 不执行任意代码，安全可控      │                                                                                                                                           
  ├─────────────┼───────────────────────────────┤                                                                                                                                           
  │ 组件目录制  │ 从预批准的组件库中选择        │                                                                                                                                           
  ├─────────────┼───────────────────────────────┤                                                                                                                                           
  │ 跨平台渲染  │ Web、Mobile、Desktop 原生支持 │                                                                                                                                           
  ├─────────────┼───────────────────────────────┤                                                                                                                                           
  │ 与 A2A 集成 │ 与 Agent-to-Agent 协议互补    │                                                                                                                                           
  └─────────────┴───────────────────────────────┘                                                                                                                                           
  渲染器支持：                                                                                                                                                                              
  ┌────────────────┬───────────────────┬────────────┐                                                                                                                                       
  │      平台      │      渲染器       │    状态    │                                                                                                                                       
  ├────────────────┼───────────────────┼────────────┤                                                                                                                                       
  │ Web            │ Lit, Angular      │ ✅ 可用    │                                                                                                                                       
  ├────────────────┼───────────────────┼────────────┤                                                                                                                                       
  │ Mobile/Desktop │ Flutter GenUI SDK │ ✅ 可用    │                                                                                                                                       
  ├────────────────┼───────────────────┼────────────┤                                                                                                                                       
  │ React          │ React Renderer    │ 🚧 2026 Q1 │                                                                                                                                       
  └────────────────┴───────────────────┴────────────┘

  7. CopilotKit AG-UI (Agent-User Interaction)                                                                                                                                              
                                                                                                                                                                                            
  定位：Agent 与用户交互的双向运行时协议                                                                                                                                                    
                                                                                                                                                                                            
  与 A2UI 的关系：                                                                                                                                                                          
  - A2UI 定义 "WHAT" — Agent 想显示什么 UI                                                                                                                                                  
  - AG-UI 定义 "HOW" — UI 如何在前后端之间流动                                                                                                                                              
                                                                                                                                                                                            
  核心能力：                                                                                                                                                                                
  - 💬 实时 Agent 聊天 + 流式传输                                                                                                                                                           
  - 🔄 双向状态同步                                                                                                                                                                         
  - 🧩 Generative UI + 结构化消息                                                                                                                                                           
  - 🛠️ 前端工具集成                                                                                                                                                                         
  - 🧑‍💻 Human-in-the-loop 协作                                                                                                                                                               
                                                                                                                                                                                            
  生态集成：

  二、Generative UI 类型分类                                                                                                                                                                
                                                                                                                                                                                            
  4. 静态 Generative UI (Static)                                                                                                                                                            
                                                                                                                                                                                            
  // AI 触发预定义组件                                                                                                                                                                      
  useCopilotAction({                                                                                                                                                                        
    name: "showWeather",                                                                                                                                                                    
    render: ({ city, temp }) => <WeatherCard city={city} temp={temp} />                                                                                                                     
  });                                                                                                                                                                                       
  - ✅ 完全可控、可预测                                                                                                                                                                     
  - ❌ 灵活性有限                                                                                                                                                                           
                                                                                                                                                                                            
  2. 声明式 Generative UI (Declarative)                                                                                                                                                     
                                                                                                                                                                                            
  // A2UI/Tambo 方式：AI 从组件注册表组装                                                                                                                                                   
  const components = [                                                                                                                                                                      
    { type: "card", component: Card },                                                                                                                                                      
    { type: "table", component: DataTable },                                                                                                                                                
    { type: "chart", component: VegaChart }                                                                                                                                                 
  ];                                                                                                                                                                                        
  - ✅ 灵活且安全，跨平台友好                                                                                                                                                               
  - ⚠️ 需要良好的组件库                                                                                                                                                                     
                                                                                                                                                                                            
  3. 完全生成式 UI (Open-Ended)                                                                                                                                                             
                                                                                                                                                                                            
  // AI 直接输出 HTML/CSS/JS                                                                                                                                                                
  const html = await ai.generate("Create a dashboard");                                                                                                                                     
  document.innerHTML = html; // ⚠️ 安全风险！                                                                                                                                               
  - ✅ 最大灵活性                                                                                                                                                                           
  - ❌ 严重安全隐患

  三、主要工具/框架                                                                                                                                                                         
                                                                                                                                                                                            
  1. CopilotKit                                                                                                                                                                             
                                                                                                                                                                                            
  - 定位：Agentic Apps 的 React 开发框架                                                                                                                                                    
  - GitHub Stars：22,000+                                                                                                                                                                   
  - 核心功能：                                                                                                                                                                              
    - useCoAgentStateRender - 基于 Agent 状态渲染 UI                                                                                                                                        
    - useFrontendTool - 前端工具调用 + Generative UI                                                                                                                                        
    - useRenderToolCall - 工具调用可视化                                                                                                                                                    
                                                                                                                                                                                            
  // 示例：基于 Agent 工具调用生成 UI                                                                                                                                                       
  useRenderToolCall({                                                                                                                                                                       
    name: "get_weather",                                                                                                                                                                    
    render: ({ args, result }) => (                                                                                                                                                         
      <WeatherCard                                                                                                                                                                          
        city={args.city}                                                                                                                                                                    
        temperature={result.temperature}                                                                                                                                                    
      />                                                                                                                                                                                    
    )                                                                                                                                                                                       
  });                                                                                                                                                                                       
                                                                                                                                                                                            
  2. Vercel AI SDK (streamUI)                                                                                                                                                               
                                                                                                                                                                                            
  - 定位：React Server Components 流式 Generative UI                                                                                                                                        
  - 状态：⚠️ RSC 开发暂停，建议迁移                                                                                                                                                         
                                                                                                                                                                                            
  import { streamUI } from 'ai/rsc';                                                                                                                                                        
                                                                                                                                                                                            
  const result = await streamUI({                                                                                                                                                           
    model: openai('gpt-4'),                                                                                                                                                                 
    tools: {                                                                                                                                                                                
      weather: {                                                                                                                                                                            
        generate: async ({ city }) => {                                                                                                                                                     
          yield <LoadingSpinner />;                                                                                                                                                         
          const data = await getWeather(city);                                                                                                                                              
          return <WeatherCard data={data} />;                                                                                                                                               
        }                                                                                                                                                                                   
      }                                                                                                                                                                                     
    }                                                                                                                                                                                       
  });                                                                                                                                                                                       
                                                                                                                                                                                            
  3. Tambo                                                                                                                                                                                  
                                                                                                                                                                                            
  - 定位：Generative UI SDK，专注组件注册                                                                                                                                                   
  - 特点：Zod schema 定义组件 props，AI 决定渲染哪个                                                                                                                                        
                                                                                                                                                                                            
  const components: TamboComponent[] = [{                                                                                                                                                   
    name: "Graph",                                                                                                                                                                          
    description: "Displays data as charts",                                                                                                                                                 
    component: Graph,                                                                                                                                                                       
    propsSchema: z.object({                                                                                                                                                                 
      data: z.array(),                                                                                                                                                                      
      type: z.enum(["line", "bar", "pie"])                                                                                                                                                  
    })                                                                                                                                                                                      
  }];                                                                                                                                                                                       
                                                                                                                                                                                            
  4. LangGraph + LangSmith Generative UI                                                                                                                                                    
                                                                                                                                                                                            
  - 定位：将 React 组件与 Graph 代码共置                                                                                                                                                    
  - 特点：组件按需加载，与 Agent Chat 无缝集成
